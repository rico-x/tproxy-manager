# TPROXY Manager v26.4.26

Изменения относительно предыдущего commit `9421731` (`Prepare release notes for v26.4.25.1`).

## Главное

- Добавлен модуль подписок Watchdog для автоматического наполнения `watchdog.links` из внешних proxy-подписок.
- Реализована поддержка Happ-подписок: capture заголовков с телефона, форма параметров подписки, ручное и таймерное обновление.
- Рабочий файл `watchdog.links` сохранён как простой список VLESS-ссылок; источники ссылок хранятся отдельно в `/etc/tproxy-manager/watchdog-subscriptions.json`.

## Watchdog subscriptions

- Добавлен helper `/usr/bin/tproxy-manager-subscriptions.lua` с командами `status`, `fetch`, `fetch-all`, `fetch-due`, `sync-links`, `exclude-link`, `capture-start`, `capture-stop`, `capture-status`.
- Добавлен блок `Подписки` перед таблицей proxy-ссылок.
- В таблицу VLESS-ссылок добавлен столбец источника: `local`, `happ N`, `json N`.
- Ссылки из подписок можно проверить, применить, переместить или исключить, но нельзя редактировать как локальные ссылки.
- Автообновление подписок подключено к циклу `/usr/bin/tproxy-manager-watchdog.sh run` через `fetch-due`.
- `SCAN_RESULT` в Watchdog теперь считает текущий список proxy-ссылок, поэтому количество обновляется после редактирования `watchdog.links` и после обновления подписок.

## Happ capture

- Добавлен временный raw HTTP capture-сервис на настраиваемом порту, по умолчанию `18088`.
- Capture-ссылка показывается в LuCI только когда сервис действительно запущен.
- Кнопка копирования ссылки сделана компактной: `💾`.
- Блок `Happ capture` свернут по умолчанию; при активном capture или просмотре последнего запроса он раскрывается.
- Добавлена кнопка `Показать последний запрос`, которая отображает сохранённые headers/body прямо в форме.
- Исправлено заполнение Happ-формы из последнего запроса с учётом разных регистров заголовков: `User-agent`, `X-HWID`, `X-Device-OS`, `X-Device-model`.
- Убрана зависимость capture от LuCI/uhttpd headers, потому что uhttpd фильтрует часть произвольных `X-*` заголовков.

## Defaults и данные

- При установке создаётся `/etc/tproxy-manager/watchdog-subscriptions.json`.
- Добавлены UCI defaults:
  - `watchdog_subscriptions_file`
  - `watchdog_happ_capture_enabled`
  - `watchdog_happ_capture_token`
  - `watchdog_happ_capture_until`
  - `watchdog_happ_capture_ttl`
  - `watchdog_happ_capture_port`
  - `watchdog_happ_capture_log`
