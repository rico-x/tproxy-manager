## Изменения

- Унифицирована расшифровка Happ-ссылок: теперь используется единый backend-модуль на Lua.
- Happ decrypt во вкладке Watchdog переведен с browser-side JS на server-side обработку.
- Ключи Happ decrypt перенесены из публичного LuCI static-каталога в `/usr/share/tproxy-manager/`.
- Удалены старые static-файлы `happ-decrypt.js` и публичный JSON с ключами.
- Упрощен `watchdog.lua`: общие parser/write/JSONC helper-функции вынесены в shared helpers.
- Happ-подписки сохраняют поддержку encrypted `happ://crypt*` URL и JSON-ответов.
- Из UI убран неактивный тип подписки `json x-ui`; JSON-ответы обрабатываются внутри Happ-подписок.
- Добавлена CI-проверка package payload на отсутствие `.DS_Store` и старых static-файлов Happ decrypt.
- Обновлена README-документация по Happ-подпискам и server-side decrypt.
